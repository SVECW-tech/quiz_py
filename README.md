Project Overview
The Online Quiz Management System is a simple, console-based Python application designed to conduct multiple-choice quizzes. It presents questions one by one, accepts user input, validates answers, and calculates the final score.
This project is ideal for beginners who want to understand Python fundamentals such as lists, dictionaries, loops, conditionals, and user input handling.
 Features
Displays multiple-choice questions
Accepts user answers (A/B/C/D)
Automatically evaluates answers
Calculates and displays final score
Easy to modify and extend
 Technologies Used
Programming Language: Python 3
IDE: Visual Studio Code (VS Code)
Execution Type: Command Line / Terminal
